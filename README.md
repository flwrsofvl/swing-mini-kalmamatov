# swing-mini-kalmamatov

Репозиторий для сдачи мини-проектов на Java Swing.

## Проект 3 — Угадай число

Программа загадывает случайное число от 1 до 100. Игрок вводит свою попытку, а приложение отвечает «больше», «меньше» или «угадал», а также подсчитывает количество использованных попыток. После победы активируется кнопка для перезапуска игры.

### Что проверяется в проекте:
* **Компоненты:** `JLabel`, `JTextField`, `JButton`
* **Логика:** Использование класса `Random` для генерации чисел
* **Архитектура:** Обработка событий через `ActionListener` и управление состояниями компонентов (включение/выключение полей ввода)
* **Надежность:** Валидация ввода и обработка ошибок (`NumberFormatException`) через блок `try-catch`

### Как запустить приложение:
1. Склонируйте репозиторий или скачайте исходный код.
2. Откройте проект в любой IDE (IntelliJ IDEA, Eclipse, NetBeans).
3. Запустите класс `src/project3/GuessNumberApp.java` через метод `main`.
